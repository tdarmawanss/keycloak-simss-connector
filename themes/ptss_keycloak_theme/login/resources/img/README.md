# Image Resources

Place your custom images in this folder.

## Required Images

### logo.png
- Your custom logo for the login page
- Recommended size: 300x80 pixels (or similar aspect ratio)
- Format: PNG with transparent background recommended
- This will be displayed at the top of the login page

### favicon.ico (optional)
- Browser tab icon
- Size: 16x16 or 32x32 pixels
- Format: ICO or PNG

## Usage

The logo is referenced in `../css/custom.css` and will automatically appear on the login page once you add the `logo.png` file to this directory.
